# Mini LLM Evaluation Framework (AI Generalist Portfolio)

## 📌 Project Overview
This project demonstrates a structured approach to evaluating Large Language Model (LLM) outputs across multiple task types, including:

- Factual queries
- Summarization
- Reasoning
- Safety & policy awareness
- Classification
- Customer support tone

Each response is evaluated using a consistent rubric to identify strengths, weaknesses, and improvement opportunities.

---

## 🎯 Objectives
- Evaluate LLM outputs systematically
- Detect hallucinations and instruction failures
- Assess tone, reasoning, and accuracy
- Build a repeatable evaluation framework
- Present findings clearly for real-world AI roles

---

## 🧪 Evaluation Rubric

Each response is scored on:

| Criteria | Description |
|--------|------------|
| Factual Accuracy | Is the information correct? |
| Instruction Adherence | Did it follow constraints exactly? |
| Logical Coherence | Is the reasoning clear and correct? |
| Tone Appropriateness | Is tone suitable for the task/audience? |
| Hallucination | Any fabricated or unsupported info (Yes/No) |

### Scoring Scale
- **1** = Major failure  
- **3** = Partial / mixed quality  
- **5** = Fully correct  

---

1. Factual Query
